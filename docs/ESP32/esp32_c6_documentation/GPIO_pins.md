# Пины ESP32-C6 (проект OpelAstraComputer)

Заметки по распиновке платы и занятым GPIO по результатам тестов и схемы.

---

## GPIO 12 и 13 — USB / Serial Monitor

**Назначение:** линия **USB D-** (GPIO 12) и **USB D+** (GPIO 13). Через них работает встроенный USB: прошивка и Serial Monitor (USB CDC), без внешнего USB‑UART.

- **GPIO 12** — выведен на плату.
- **GPIO 13** — на плату не выведен.

**Эти пины трогать нельзя.** Использование их как обычных GPIO (OUTPUT/INPUT) ломает USB: пропадает связь с Serial Monitor и прошивка по USB. В коде не назначать на них периферию и не переключать вручную.

---

## GPIO 22 — подсветка дисплея

**Назначение:** в проекте используется как **подсветка (backlight)** дисплея ST7789 (`EXAMPLE_PIN_NUM_BK_LIGHT` в `Display_ST7789.h`).

Тест диагностики GPIO показал: при переключении GPIO 22 дисплей мигает (подсветка включается/выключается). То есть пин 22 занят дисплеем.

- **GPIO 22** — на плату не выведен (подключен к дисплею на плате).

Для своих целей этот пин использовать нельзя — он уже занят подсветкой.
